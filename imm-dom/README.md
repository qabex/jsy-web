# qabex/web-jsy/imm-dom

Template for JSY and [js-imm-dom](https://github.com/shanewholloway/js-imm-dom) web development:


```bash
npm init from-gh qabex/web-jsy/imm-dom some-project
```

## Usage

```bash
./finish-init.sh
npm run build # or npm run watch
npm run serve
```
